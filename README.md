# Reel Flow

Reel Flow is a modern short-video social media application inspired by Instagram Reels and TikTok. It provides a clean and responsive user interface with Firebase Authentication and a vertical video feed.

## Features

- User Authentication
- Google Sign-In
- Short Video Feed
- Responsive UI
- Modern Design
- Firebase Integration

## Tech Stack

- React
- JavaScript
- Firebase
- HTML
- CSS

## Project Status

This project is currently under development. Some backend features require Firebase Firestore configuration and billing.
