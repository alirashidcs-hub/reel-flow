## 📱 Overview

Reel Flow is a modern short-video sharing platform that enables users to upload, watch, and interact with short videos. The project is inspired by TikTok and Instagram Reels and focuses on providing a clean, responsive, and engaging user experience.

---

## ✨ Features

- 🔐 Email Authentication
- 🔑 Google Sign-In
- 👤 User Profiles
- 🎥 Vertical Video Feed
- ❤️ Like System
- 💬 Comments
- 🔍 Search Users & Videos
- 📈 Trending Section
- 💌 Messaging UI
- 🔔 Notifications
- ⚙️ Admin Dashboard
- 📱 Responsive Design

---
